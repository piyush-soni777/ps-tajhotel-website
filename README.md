# 🏨 Taj Hotel Website

Taj Hotel Website is a modern and responsive hotel booking UI built using HTML, CSS, and JavaScript.  
It showcases a luxury hotel experience with elegant design, smooth animations, and user-friendly navigation.

## ✨ Features
- 🏨 Elegant and luxury hotel UI design  
- 📱 Fully responsive (Mobile + Desktop)  
- 🛏️ Room showcase and details section  
- 📸 Image gallery for hotel preview  
- 📞 Contact / booking section  
- ⚡ Smooth animations and navigation  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## ⚙️ How It Works
- Users can explore hotel rooms and services  
- Navigate through sections like rooms, gallery, and contact  
- Smooth scrolling and interactive UI enhance user experience
